# Установка D1 Control

## Быстрая установка

```bash
# Запустите скрипт установки
./setup.sh
```

Скрипт автоматически:
- Установит все необходимые пакеты (git, cmake, Qt5)
- Скачает и установит Unitree SDK2
- Соберёт проект

## После установки

```bash
# Запустите систему управления
./start.sh
```

## Настройка сети

Рука использует IP-адрес `192.168.123.100`.

Настройте ваш компьютер:
```bash
sudo ip addr add 192.168.123.10/24 dev eth0
```

## Проверка установки

```bash
./setup.sh --verify
```

## Полная переустановка

```bash
./setup.sh --clean
```
